Welcome to Last Modified Timestamp

Thank you for installing Last Modified Timestamp! This extension helps you find when a webpage was last updated.

For optimal experience, we recommend pinning this extension to your toolbar.
1
Click the extensions button in your Chrome toolbar
Find the Extensions puzzle icon in the top-right corner of Chrome
2
Pin the Last Modified Timestamp extension
Click the pin icon next to "Last Modified Timestamp" to keep it visible in your toolbar
3
That's it!

The extension will automatically check for timestamps on each page you visit. You'll see:

  • A temporary overlay in the top-right corner showing the timestamp
  • The extension icon will show a green checkmark if a timestamp is found
  • Hover over the icon to see the timestamp in a tooltip

You can also click the extension icon anytime to recheck the current page.
